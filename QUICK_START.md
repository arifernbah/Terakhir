# 🚀 **QUICK START GUIDE**

## 📋 **Setup dalam 5 Menit**

### **1. Install Dependencies**
```bash
pip install -r requirements.txt
```

### **2. Configure API Keys**
Edit file `.env`:
```env
API_KEY=your_binance_api_key_here
API_SECRET=your_binance_secret_here
TELEGRAM_TOKEN=your_telegram_bot_token
TELEGRAM_CHAT_ID=your_chat_id
```

### **3. Jalankan Bot**
```bash
python3 main.py
```

## 🔑 **API Setup**

### **Binance API:**
1. Login ke [Binance](https://binance.com)
2. Account → API Management
3. Create API Key → Enable Futures Trading
4. Copy API Key & Secret

### **Telegram Bot:**
1. Chat [@BotFather](https://t.me/botfather)
2. `/newbot` → Buat bot baru
3. Copy bot token
4. Chat [@userinfobot](https://t.me/userinfobot) untuk chat ID
